n-jin

Node.js Web Crawler & Search Engine Base

Project Overview

Purpose: A lightweight and efficient Node.js Web Crawler designed to crawl websites starting from a base URL, extract internal links, and recursively process pages.

How it works: This project demonstrates search engine and indexing system fundamentals. It normalizes URLs to avoid duplicates and tracks the number of times each page is found, making it an excellent foundation for an SEO analyzer or a full search engine.

Key Features

Built as a pure Node.js application, it prioritizes speed, efficiency, and a deep understanding of core web crawling concepts.

  • Recursive Crawling: Efficiently crawl all linked pages within a single website starting from a designated base URL.
  • Link Extraction: Parse HTML safely to extract all valid internal links while ignoring external properties.
  • URL Normalization: Built-in safeguards to normalize diverse URL structures, comprehensively preventing recursive duplication and infinite loops.
