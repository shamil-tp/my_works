Blogify

Modern Collaborative Blogging Platform

Project Overview

Purpose: To streamline the way developers write and share articles by focusing on structured content delivery, moving away from standard plain-text storage to a robust, custom-tailored data architecture.

How it works: Authors use a custom Slate.js editor to generate a clean JSON tree representing the document structure, which is then dynamically rendered on the frontend using Tailwind-styled React 19 components.

Key Achievements & Contributions

Collaborated as a core developer to engineer the platform's robust backend API, custom editor integration, and modern frontend architecture.

  • Custom Editor Engine: Implemented a custom Slate.js engine to serialize content into structured JSON formats.
  • Backend Pipeline: Built secure Express.js routes with Zod validation, JWT authentication, and optimized MongoDB queries.
  • Modern Frontend: Developed a responsive and high-performance React 19 UI powered by Tailwind CSS 4.0 and Vite.
