Skylark

Premium Rent-A-Car Platform

Project Overview

Purpose: To offer customers an easy, fast, and reliable way to book cars online while giving administrators powerful tools to manage fleet, track issues, and analyze revenue.

How it works: Users browse a luxury fleet (BMW, Benz), pick dates, and securely pay via Razorpay. Admins use a dedicated portal for fleet management, deep analytics, and customer ticket resolution.

Key Achievements & Contributions

As a core developer in the 3-member team, I engineered the entire Admin Panel, Vue architecture, and massive backend integrations.

  • Vue Architecture: Built the entire Router structure, Layout components (AdminLayout, AuthLayout, MainLayout), and Pinia state stores.
  • Backend Services: Integrated Razorpay APIs, complex Mongoose schemas (Car, User, Issue, Booking), OTP Mailer workflows, and robust JWT middlewares.
  • Frontend Pages: Single-handedly crafted the Admin interface (Dashboard, Users, Issues, Reports), Authentication pages, and intricate dynamic Booking logic.
